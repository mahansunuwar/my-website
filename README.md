# Mahan Sunuwar Portfolio (TypeScript + Node.js)

A professional cybersecurity portfolio website built with:
- Node.js + Express
- TypeScript (backend and frontend)
- Modern responsive UI

## 1) Install dependencies

```bash
npm install
```

## 2) Run in development

```bash
npm run dev
```

Open `http://localhost:3000`

## 3) Build for production

```bash
npm run build
npm start
```

## Where to edit your content

Update your portfolio details in:

`src/data/portfolio.ts`

You can change:
- Name, role, and summary
- Project list and impact
- Achievements
- Certifications
- Contact links

## Project structure

- `src/server.ts` - Express server and API endpoint
- `src/data/portfolio.ts` - Portfolio content
- `src/client/main.ts` - Frontend rendering logic
- `public/index.html` - Page layout
- `public/styles.css` - Professional cybersecurity styling
